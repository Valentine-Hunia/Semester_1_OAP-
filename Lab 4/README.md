Інструкція з запуску застосунку

Встановлення залежностей(Якщо їх нема)
npm install

Запуск серверу бекенду
Відкрити термінал у папці з бекендом
ввести команду npm run dev
Сервер буде запущено на порті 3000 (http://localhost:3000)

Запуск серверу фронтенду
Відкрити термінал у папці з фронтедом
Скомпілювати код за допомогою команди npx tsc
Запустити локальний сервер бекенду (без кешування)
Команда: npx http-server -p 5500 -c-1

Відкрити у браузері адресу http://localhost:5500
